## 🎯 核心技术栈

### 前端
- **框架**: Next.js 15 (App Router)
- **语言**: TypeScript 5
- **样式**: Tailwind CSS 4
- **UI组件**: shadcn/ui
- **视频**: Video.js
- **状态**: Zustand + SWR
- **图标**: Lucide React

### 后端
- **运行时**: Node.js 18+
- **框架**: Next.js API Routes
- **数据库**: PostgreSQL
- **ORM**: Prisma
- **认证**: JWT
- **AI**: Google Gemini

### 部署
- **平台**: Vercel
- **数据库**: PostgreSQL (云端)
- **CDN**: 视频签名URL
